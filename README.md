# La Galardonada

Recreación web desarrollada en React y TypeScript para la empresa La Galardonada, especializada en servicios de limpieza profesional en la Comunidad de Madrid.

## Descripción

Este proyecto consiste en la implementación de una web corporativa moderna, responsive y optimizada, incluyendo diferentes páginas de servicios como:

- Limpieza de sofás
- Limpieza de colchones
- Limpieza de alfombras
- Limpieza de cortinas
- Limpieza de edredones
- Restauración de alfombras

El objetivo ha sido replicar la estructura, diseño y experiencia de usuario de la web original utilizando tecnologías modernas del ecosistema React.

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/tomasvillani/lagalardonada.git
```

Entrar en el proyecto:

```bash
cd lagalardonada
```

Instalar dependencias:

```bash
npm install
```

Iniciar el entorno de desarrollo:

```bash
npm run dev
```

## Objetivos del proyecto

- Desarrollar una aplicación React estructurada y mantenible.
- Aplicar buenas prácticas de maquetación y responsive design.
- Mejorar la reutilización de componentes y estilos.
- Simular un entorno de desarrollo real para una web corporativa.
